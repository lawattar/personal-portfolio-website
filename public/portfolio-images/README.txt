Place project media subfolders here for modal slideshows.

Folder convention:
- public/portfolio-images/<project-key>/

File convention inside each project folder:
- 1.jpg, 2.jpg, 3.png, 4.mp4, ...

Supported image types:
- .jpg .jpeg .png .webp

Supported video types:
- .mp4 .webm .mov

Examples:
- public/portfolio-images/lincoln-electric/1.jpg
- public/portfolio-images/sr26-steering-wheel/1.png
- public/portfolio-images/rugby/1.mp4

Current project keys used by the site are defined in projects.js under `key`.
